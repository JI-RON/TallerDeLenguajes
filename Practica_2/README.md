## Objetivos

* Resolver problemas más complejos utilizando estructuras de datos adecuadas.
* Aplicar correctamente el uso de Jupyter Notebook.
* Desarrollar soluciones considerando dependencias y buenas prácticas.

## Descripción del código

*  **Ejercicios_1_9.ipynb** es una Jupyter Notebook con la resolución de los ejercicios 1-9. Para ejecutarse importa los módulos procedentes de
    *  **helpers_1_9.py**, que contiene las funciones y módulos necesarios.
*  **Ejercicios_10.ipynb** es una Jupyter Notebook con la resolución del ejercicio principal del práctico (Ejercicio 10). Para ejecutarse importa los módulos procedentes de
    *  **helpers_10.py**, que contiene las funciones y módulos necesarios.

## Instrucciones de instalación

*  Descargar proyecto desde repositorio de GitHub:
*  Crear entorno virtual.
    *  Abrir terminal cmd.
    *  Navegar hasta la carpeta del proyecto descargado, "Proyecto 2026', utilizando el comando 'cd'.
    *  Una vez localizado allí, crear entorno virtual utilizando el comando 'python3 -m venv .venv'.
*  Localizado en la carpeta donde se encuentra creado el entorno virtual, activarlo utilizando el comando '.venv\Scripts\activate.bat'.
*  Instalar las dependencias contenidas en **requirements.txt** a través de cmd con el comando 'pip install -r requirements.txt'.
*  Ejecutar Jupyter con el comando 'jupyter-lab' y proceder a comprobar la resolución de los ejercicios.
    *  Para ejercicios 1-9, abrir el archivo **Ejercicios_1_9.ipynb**
    *  Para ejercicio 10, abrir el archivo **Ejercicio_10.ipynb**
    *  Ejecutar celdas con el botón 'Run All' para comprobar resultado.
        *  ACLARACIÓN: En algunos ejercicios se solicita el ingreso de valores por teclado.
*  Cerrar Jupyter.
*  Cerrar el entorno virtual con el comando '.venv\Scripts\deactivate'.

# Taller de Lenguajes 2026 (Python) - Actividad 2

## Descripción

Este proyecto contiene la resolución de una serie de ejercicios prácticos en Python orientados al uso de estructuras de datos, procesamiento de información y buenas prácticas de programación.

Los ejercicios se encuentran organizados en notebooks de Jupyter y utilizan módulos auxiliares para encapsular la lógica.

## Objetivos

* Resolver problemas más complejos utilizando estructuras de datos adecuadas.
* Aplicar correctamente el uso de Jupyter Notebook.
* Desarrollar soluciones considerando dependencias y buenas prácticas.

## Descripción del código

*  **Ejercicios_1_9.ipynb** es una Jupyter Notebook con la resolución de los ejercicios 1-9. Para ejecutarse importa funciones definidas en
    *  **helpers_1_9.py**, que contiene las funciones y módulos necesarios.
*  **Ejercicio_10.ipynb** es una Jupyter Notebook con la resolución del ejercicio principal del práctico (Ejercicio 10). Para ejecutarse importa funciones definidas en
    *  **helpers_10.py**, que contiene las funciones y módulos necesarios.

## Instrucciones de instalación

*  Requisitos:
    - Python 3.12 instalado.
    - pip instalado.
*  Clonar o descargar el repositorio de GitHub (https://github.com/JI-RON/TallerDeLenguajes/tree/main/Practica_2):
*  Crear entorno virtual.
    *  Abrir una terminal.
    *  Navegar hasta la carpeta del proyecto descargado, "Proyecto 2026".
    *  Una vez localizado allí, crear entorno virtual utilizando el comando "python3 -m venv .venv".
*  Localizado en la carpeta donde se encuentra creado el entorno virtual, activarlo:
    - Windows: .venv\Scripts\activate
    - Linux / Mac: source .venv/bin/activate
*  Instalar las dependencias contenidas en **requirements.txt** a través de cmd con el comando "pip install -r requirements.txt".
*  Ejecutar Jupyter con el comando "jupyter-lab" y ejecutar los notebooks para visualizar los resultados.
    *  Para ejercicios 1-9, abrir el archivo **Ejercicios_1_9.ipynb**
    *  Para ejercicio 10, abrir el archivo **Ejercicio_10.ipynb**
    *  Ejecutar celdas con el botón "Run All" para comprobar resultado.
        *  ACLARACIÓN: En algunos ejercicios se solicita el ingreso de valores por teclado.
